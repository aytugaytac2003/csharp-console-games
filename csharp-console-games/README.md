# C# Console Games & Fundamentals

This repository contains multiple console-based projects developed with C# to practice fundamental programming concepts and object-oriented programming (OOP).

The projects were built as part of my learning journey while studying C# and preparing for backend development with .NET.

---

## Projects Included

###  Number Guessing Game
A simple game where the computer selects a random number and the player tries to guess it.

**Concepts used:**
- Variables
- Loops (while)
- Conditional statements
- Random number generation
- User input validation

---

###  Quiz Game
A console-based quiz application with multiple-choice questions and score tracking.

**Concepts used:**
- Classes & objects
- Lists and collections
- Loops (foreach, for)
- Methods
- Basic OOP principles

---

###  Mini RPG Game
A turn-based console RPG game where a player fights against an enemy.

**Concepts used:**
- Object-oriented programming (OOP)
- Inheritance
- Encapsulation
- Game loop logic
- Randomized damage system
