# Quiz Game

A console-based quiz application developed with C# to practice object-oriented programming concepts.

## Description
The game asks multiple-choice questions and calculates the player's score based on correct answers.

## Features
- Multiple-choice questions
- Score tracking
- Console-based user interaction

## Technologies
- C#
- .NET Console Application

## What I Learned
- Creating and using classes
- Working with lists and collections
- Applying basic OOP principles
- Looping through structured data
