#  Mini RPG Game

A simple turn-based RPG game built with C# to demonstrate object-oriented programming and game logic.

## Description
The player fights against an enemy using attack and heal actions.  
The game continues until one character's health reaches zero.

## Features
- Player vs Enemy combat
- Turn-based gameplay
- Health and attack system
- Randomized damage

## Technologies
- C#
- .NET Console Application

## What I Learned
- Object-oriented programming (OOP)
- Inheritance and encapsulation
- Game loop and state management
- Designing reusable classes
